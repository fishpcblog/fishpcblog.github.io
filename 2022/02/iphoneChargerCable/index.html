買iPhone充電線時要注意的事項[推薦給大家]

我之前買的一條200元的iPhone充電線頭斷了，整個接頭跟線快要分開了。
如果繼續用下去我會覺得很危險，於是我沒有考慮很多就直接再去買了一條200元的充電線，結果這一條充電線帶給我很不方便，於是我上網找相關資料，首先我要考慮的點有4點

  1. 挑有MFI認證的週邊配件
  2. 充電線長度
  3. 充電接頭規格
  4. 充電線材質

挑有MFI認證的IPhone週邊配件

市面上有很多在賣iPhone充電線，我們要買的時侯要挑選有MFI認證的iPhone充電線。什麼是MFI認證？MFI全名是Made for iPhone| iPod | iPad 縮寫。簡單說就是Apple公司制定了MFI認證計劃讓想製作Apple產品的周邊配件公司申請，申請通過後所核發下來的認證。

所以我們購買充電線時可以找產品外包裝上有沒有Made for iPhone | iPod ｜iPad 的圖標，然後上這個 MFi Licensed Accessories 網站查詢做雙重確認，一定要做雙確認哦，這樣是避免買到買到仿冒的，購買有MFI認證的好處是確保iphone充電線相容lightning週邊設備，品質至少比沒MFI認證的好,然後避免iPhone充電線插入手機沒反應。

充電線長度

我們要買充電線時要考慮充電線要使用在那些情境上，那這些使用情境我的充電線夠長嗎？

  • 你要接電腦USB孔傳資料時，你的電腦USB孔與桌子的距離(接電腦時會不會有時侯操作手機)
  • 你要接充電器充電時，你牆壁上的110V插座與桌子的距離或者與床的距離

這次我要買的iPhone充電線是要用在電腦與手機傳資料用，那我要怎麼量這個距離呢？答案是拿一條塑膠繩充當iPhone充電線去測量
我們要準備材料有

  • 塑膠繩長度約雙手張開長度
  • 電火布(這種粘性比較好，一般膠帶也可以)

量電腦主機與在桌上操作手機距離

我們可以在你要插的USB孔上，塑膠繩先預留USB頭的長度然後用電火布或者膠帶把塑膠繩貼住，如下圖

然後塑膠線的另一頭拉到桌上接下來用手把塑膠線跟手機握住，然後上下左右移動看塑膠線會不會線太短或太長，太短就加長塑膠線長度，太長就減少塑膠線長度，我們取一個適當的長度。如下圖

接下來我們將塑膠繩取下來用捲尺或者尺量出塑膠繩的距離，我的電腦USB孔到桌上量出來結果是1.38m,所以我要找儘量大於1.38m長度的iphone充電線，雖然市面上販賣的長度大多是1米的跟2米的不過也有其他長度,如果事前沒考慮到買回來的線太短可是會不合用的，我就是其中一個。

如果要量充電器到桌上距離或者到床上距離做法也是跟上述做法一樣，用膠帶把塑膠線貼在充電器側邊拉至桌上看距離多少。

充電線接頭規格

一般Iphone手機充電線的接頭分爲

  • USB Type C(USB-C )to lightning
  • USB(Type A) to lightning

USB(Type A)接頭外型是傳統USB頭跟USB隨身碟接頭一樣。
USB Type C接頭比較小，外形像是操場跑道形的扁頭，扁頭中間有凹槽，它的體積略大mirco-usb，接頭不分正反面，如果要插入手機或電腦可以用正面插也可以用反面插。

如果i8以上手機想要使用PD快充則必須買USB- C（type C）to lightning充電線才行，如果使用USB(type A)to lightning則是不支援的。

我們要購買時這些接頭規格都是考慮因素，像我拿i6S不支援PD快充所以要買USB(type A)to lightning充電線，齁～真想換新機啦～～。

充電線材質

接下來我們說說充電線材質，市面上有很多琳琅滿目iphone充電線，我覺得它們的差異在於用料，譬如充電線的內部銅線的線徑、阻擋emi電磁波的錫箔紙、絕緣體、外部的接頭強化設計、線的抗折疊、抗拉扯材料比如電線最外層覆上一層編織布等等…，這些都是我們考慮的因素。
如果是我的話，我會選接頭有強化的設計(因為我上一條是接頭斷了)，然後充電線有抗拉扯的材質為佳，銅線比較粗的尤佳，這樣可以減少銅線斷掉的機會避免接觸不良或短路。

重點整理

今天我介紹了買iphone 充電線時我沒考慮的事項，我自己也是上了一課，接下來我們複習一下要買Apple產品的充電線可以注意底下這幾點

  1. 挑有MFI認證的週邊配件
  2. 充電線長度
  3. 充電線接頭規格
  4. 充電線材質

綜合以上資訊，我決定重新買一條充電線了。

買3C產品就是多比價多了解3C產品這樣才不吃虧，也不要買價格太低的充電線，最後希望這篇文章能幫助到你買iphone充電線時要注意的事項，如果這篇文章對你有幫助請記得按讚並分享給有需要買iphone充電線的人。
