如何最小化安裝Fedora系統

Fedora Workstation 使用快一年至近至近體驗還不錯，最近在想Fedora有沒有可能跟arch那樣從最基礎Linux系統一點一滴安裝自己所需的軟體就好，沒想到Fedora也有提供這種安裝方法。那如何最小化安裝Feodra系統請繼續看下去。

下載Fedora Everything版

用瀏覽器打開此網站，下載Fedora Everything 版ISO檔

Fedora Everything 版ISO檔寫入隨身碟

安裝gnome-disk

打開檔案總管瀏覽到Fedora Everything 版ISO檔目錄下，滑鼠選到該檔案按右鍵，選[Open With Image writer]，選擇要寫入ISO檔隨身碟，然後滑鼠按[開始恢復]，等待寫入完成。

BIOS設定

首先電腦開機等待聽到主機發出嗶一聲，請狂按主機板製造商提供的熱鍵進入BIOS，將Security boot 關閉並將CSM設成自動。自動的話就是系統自動偵測啟動裝置是UEFI或者是MBR和附加裝置。

進入Fedora安裝程式畫面

接下來電腦USB孔插入Fedora Everything 系統隨身碟，然後電腦重開機，等待聽到主機發出嗶一聲，請狂按主機板製造商提供的熱鍵進入啟動選單，再來選擇有UEFI開頭的雖身碟名稱後按Enter，等待進入開機選單後選擇[Test this media & install Fedora36] 後按Enter，
系統會驗證ISO檔是否完整後會進入安裝程式畫面。

接下來用快速操作指南方式說明格式如下：
達成任務或者目前畫面標題：操作動作

  • 歡迎使用Fedora 36：選擇[中文]➝[繁體(臺灣)]➝按[繼續]

    安裝摘要

    Fedora安裝摘要畫面要設定的有
  • 本地化設定
  • 軟體
  • 系統
  • 用戶設定

本地化設定

鍵盤配置：選擇中華民國(臺灣語)➝按[減號按鈕]➝選擇英文(英文(美式))➝按[加入按鈕]➝按[完成按鈕]
語言支援：選擇English➝打勾English(United States)➝按[完成按鈕]

軟體

安裝來源：選擇最近的鏡像站
選擇軟體：選擇[最小安裝] ➝打勾[標準選項]
➝按[完成]

系統

安裝目標位置：選擇[進階自訂(Blivet-GUI)] ➝按[完成按鈕]

Blivet-GUI磁碟分割

總共要分割4個分割區

  • 500MB efi分割區
  • 80G root 目錄
  • 100G home 目錄
  • 1G swap

500MB efi分割區：按[加號按鈕]➝裝置類型選分割區➝大小輸入500➝單位MB➝檔案系統選efi➝標籤輸入efi➝掛載點輸入/boot/efi

80G root 目錄：按[加號按鈕]➝裝置類型選Btrfs儲區➝大小輸入80➝單位GB➝名稱欄位輸入fedora_root➝掛載點輸入/
100G home 目錄：按[加號按鈕]➝裝置類型選Btrfs儲區➝大小輸入100➝單位GB➝名稱欄位輸入fedora_home➝掛載點輸入/home
1G swap：按[加號按鈕]➝裝置類型選分割區➝大小輸入2.4➝單位GB➝檔案系統選swap➝標籤輸入fedora_swap

全部設定完成，滑鼠按螢幕左上角[完成按鈕]按[加號按鈕]

網路與主機名稱

主機名稱輸入super(依照你想輸入名稱)➝按[套用按鈕]➝滑鼠按螢幕左上角[完成按鈕]

開始安裝系統

滑鼠按螢幕右下角[開始安裝按鈕]，使用這種方式必須有要網路才能安裝

Fedora基礎系統安裝完後，電腦重開機後登入系統就可以一點一滴安裝自己所需軟體及桌面環境了。

重點整理

以上是我今天整理如何最小化安裝Feodra系統文章，使用Fedora Everything 版必須有要網路才能安裝Fedora系統。
